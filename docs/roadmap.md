# Enterprise Decision Intelligence Engine (EDIE)

## Project Goal

Build a multi-agent AI decision intelligence platform where specialized AI executives analyze, debate, and recommend strategic business decisions.

The platform will simulate an enterprise boardroom and generate executive-level decision reports based on multiple perspectives.

---

# Phase 1 - Foundation

## Objectives

* Setup repository structure
* Setup NestJS backend
* Setup PostgreSQL
* Setup Redis
* Setup Docker
* Setup Swagger

### Deliverables

* API Gateway
* Projects Module
* Assessments Module
* Docker Compose
* Basic Authentication

# MVP Definition

A successful MVP must:

* Accept a business problem
* Execute at least 3 AI agents
* Generate debate outputs
* Produce final recommendation
* Display workflow visually
* Run locally using free models

No paid APIs required.
